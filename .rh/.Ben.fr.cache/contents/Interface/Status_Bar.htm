Status Bar Displays some basic information for the current selected session. 1. Session info. Date and time of the current session. 2. Nominee Count. Total count of nominations in the current session. 3. Entrant Count. Total count of entrants in the current session. 4. Status and message section. General status and warning messages posted by the SCM application. 5. Resize the form widget. Resize the SCM's main form.
