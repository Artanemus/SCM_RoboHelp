SCM_BuildMeAClub. A run-it-once utility to get your SwimClubMeet started. This builds all the necessary tables and system data onto the MSSQL Server. It will ask you for the SCM build file, an attachment that was emailed to you on purchase of the product. There a five steps: Step 1: Introduction Step 2: Prerequistes Step 3: Server Details Step 4: SCM File Location Step 5: Go Build me a Database
