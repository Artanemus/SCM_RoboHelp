Import CarnivalMeet...

TBA. Import your members event and racetime data from the recent carnival meet. The imported data will appear as a Session within SwimClubMeet. The session date will match the CarnivalMeet's date.

Export CarnivalMeet...

TBA. Export your members details (name, gender, DOB) and personal best across all raced events in prepartion for import into CarnivalMeet.

Exit

Closes the application. There is no need to save. Data is continuously written to the MSSQL server.
