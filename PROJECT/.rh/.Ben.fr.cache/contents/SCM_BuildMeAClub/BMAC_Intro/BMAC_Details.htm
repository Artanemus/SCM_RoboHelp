This special tool creates the SwimClubMeet database on SQLEXPRESS. It ensures that you have SQLEXPRESS by asking you to login into the SQL server. If successfull it'll display the button 'Build Me A Club'. When you press this button it will first make some checks. If you already have a SwimClubMeet database on you SQL server, it'll abort. The application has a large number of SQL scripts that it will run on the SQL server. SCM_BuildMeAClub uses the sqlcmd.exe commandline utility when processing these scripts. SQLcmd.exe is installed by default when use first install SQLEXPRESS. The application will check that you have SQLcmd.exe installed. If it can't find the utiltiy it will abort. As it runs it will display it's progress and display information in the application's text window. As it runs, it will create the SwimClubMeet database, create all of the tables, procedures and scalar functions. Then it will populate some of the tables with 'starter' data. It will create a user, scmAdmin. A set of roles : SCM_Administrator, SCM_Marshal and SCM_User. It will assign permissions for each role on each table. As the SQL server performs it's given tasks it will log it's progress to a file created in the user's document folder. That file is named SCM_BuildMeAClub.log.. If the SQL server encounters any errors, it will log the errors to this file. If the command line utility SQLcmd.exe encounters an error, it will log the error to the SCM_BuildMeAClub text windows. Typically the whole build process is over within a minute or two. NOTE: the term 'SQLEXPRESS' and 'SQL server' are identical in meaning.
