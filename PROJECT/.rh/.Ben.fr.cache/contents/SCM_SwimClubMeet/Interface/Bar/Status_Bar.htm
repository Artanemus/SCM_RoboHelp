Status Bar Displays some basic information for the current selected session. 1. Session info. Date and time of the current session. 2. Nominee Count. Total number of nominations by members across the entire session. 3. Entrant Count. Total number of nominees that have been given a swimming lanes (wit. entrants) across the entire session. If the number of nominee equals the number of entrants, then nobody missed out on swimming their chosen races. 4. Text message section. This space is reserved by SCM to display the occasional status message. Warnings and error messages are typically displayed in dialogue boxes. 5. Resize the form widget. Resize the width and height of SCM's form.
