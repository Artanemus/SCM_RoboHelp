Lane Buttons This section is closely tied to the lane grid. For more details on each button's actions, view the lane grid topic ... link. These buttons will be grayed (disabled) if the heat has been raced or closed or if the session has been closed. A short summary of the lane buttons. Move entrant up. When you hover over this button a hint message will appear.Move the entrant up a lane (CTRL+Up). This will shift the entrant (or an empty lane) up, not just in the current heat but across into other heats. Move entrant down. When you hover over this button a hint message will appear. Move the entrant down a lane (CTRL+Down). Swap Lanes. When you hover over this button a hint message will appear. Opens the swap lane dialogue. Click 'Swap Lanes' to change the selected heat's stack order. Empty Lane. When you hover over this button a hint message will appear. Empty the lane. The member remains nominated. (CTRL+X). Clicking the button, displays the confirmation dialogue box. Answer yes to remove the entrant from the lane. Remove nomination and empty lane. When you hover over this button a hint message will appear. Remove nomination and empty the lane. (CTRL+Del). Clicking the button, displays the confirmation dialogue box. Answer yes to fully scrub lane and nominee.
