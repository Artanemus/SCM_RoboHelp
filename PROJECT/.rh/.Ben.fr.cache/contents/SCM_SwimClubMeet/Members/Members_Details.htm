Members Details SCM stores the following information about each swimming club member. Summary description for each field ... Membership ID Typically a registration number given by the main swimming body or this Swim Club. Or a short digit number used in self nomination. (ie. when using the nominate support app). Swimming Club Assigned by SCM during installation - this field is read only. First Name Christian and middle names go here. Spaces are accepted. Last Name   Club member's surname. Dual surnames should use the hypen character to seperate. Spaces  are not accepted. Reports pre-process dual surnames and are displayed correctly.  Date of Birth (DOB) Gender Gender. Is Active The member is participating in the club for the current swimming season. Disabling this checkbox effectively archives the member. In-active members are excluded from nominating for events.. Membership Type Use the drop-down combobox an select a membership type. What types appear is set in the Membership type tool dialogue. Email Address Used for mail-outs and website identification. Membership Due Expire date? House The house this member is assigned too.      Contact numbers  If you would like to place contact information then use this table. The contact number accepts alpha-numerical. More than one number can be entered. CNTRL-DELETE to delete a selected contact number CNTRL-INSERT to add a new contact number. Or press TAB at end of the last line to create a new record. Use the contact type drop-down combobox to select Mobile, Home or Business
