Heat Buttons

This section is very closely tied into the heat grid. Found here ... link.

 

A brief summary of the heat buttons.

 

 

Sort or move buttons.

Select the heat to move then press either ARROW-UP or ARROW-DOWN buttons to change it's stack order.  These buttons will be grayed (disabled) if the heat has been raced or closed or if the session has been closed.

 

Toggle the heat status.

Click this button to toggle between open, raced and closed. This button will be grayed (disabled) if the heat has been raced or closed or if the session has been closed.

 

New Heat

When this button is pressed a new heat and all it's lanes are created.  This button will be grayed (disabled) if the session has been closed.

 

Delete Heat

Delete the heat. You are asked to confirm if you wish to delete the heat. This button will be grayed (disabled) if the heat has been raced or closed or if the session has been closed. There is no undo!

 

Auto-Build Heats

Clicking the Auto-Build Heats button will open the auto-build heats dialogue. This button will be grayed (disabled) if the session has been closed.

All the information on Auto-Build Heats is found here ... link.

 

View, marshal's sheet.

Clicking this button displays version 1 of the marshal's sheet  CNTRL-Click to display version 2. In either view, you have the option to print it.

Examples of the marshal's report ... link.

 

View the time keeper's sheet.

Clicking this button displays version 1 of the time keeper's sheet for the current selected heat. CNTRL-Click to display version 2. In either view, you have the option to print it. 

Examples of the time keeper's report ... link.

 

Print set.

Clicking this button will show a dialogue asking you to select a printer to send a marshal's sheet and ALL of the event's time keeper's sheets. Version 1 is the layout that is used. Printing start's imediately you press ok on the print selection dialogue. You are not shown a preview of the sheets. This is a quick method to get sheet's printed and distributed.

 

View the heat report.

Clicking this button displays version 1 of the heat report  CNTRL-Click to display version 2 of the heat report. In either view, you have the option to print it.
