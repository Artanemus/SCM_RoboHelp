Your first club night with SCM.

Here's an outline of what needs to be done to get started. 

  1. You need swimmers. Use the Club Members button and add your members.

    TIP: Get yourself comfortable, this is going to take some time. Good news - you only do this once. Minimum requirements are FirstName, LastName, DOB and gender.

     
  2. Add a new "Session" in the "Session Table". With that session selected ...

    TIP: A short note makes the session memorable. 'Clear skies and bar-b-que'.

     
  3. Add your events into the "Events Table" by entering "Distance" and "Stroke". 

    TIP: If you need to describe your events. Click the expander button to expand the event grid. The 'description' column is now visible. Any text entered here will be seen by everyone and displayed in every report.

     
  4. Select the next tab-sheet - Nominate. Here you select the swimmer then check the events they will be swimming.

    TIP: Need to know if they are 'Qualified' to swim the event. Toggle the 'Qualified' button  on.

     
  5. Select the next tab-sheet - Heats. Press the Build Heats button for each event. Be aware, your database, at this stage, is totally empty. So your swimmers will not be 'matched-up' correctly. (See the special note below.) 

    TIP: There's a 'build all heats' in the menu bar.

     
  6. You are ready to print the marshal and time-keepers' forms.

    TIP: If the 'print-set' button isn't enough, there's a 'print all marshal and timekeeper forms' in the menu bar. But beware, the windows print spooler is probably turned on and that's a lot of paper. 

     
  7. When a heat has been swum - enter the swimmers' times into the "race time'' column.

    TIP: Arrow down to move to the next race time.

     
  8. When the session is completed, print the session report to PDF and email.

    TIP: Check-out the 'qualified swimmers report' in tools >> qualification times.

That it. 

Here's a link to a YouTube Video that takes you through the above process.

 

Special note:

If you want to 'kick-start' your TimeToBet algorithm (you have just installed the application and you have no race data) add a new session that is dated prior to the new swimming season. Populate that session with every type of event. Then fill selected events with the personal best race-times for each of your members. That way when the very first SCM club night is played ... entrants are correctly placed. This also a useful technic for new members that join late in the season.
