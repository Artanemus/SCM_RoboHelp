Members Index

Create a new member, enter details, track attendance, visualize a member's performance and more.

MEMBERS 5 X TAB-SHEETS AND MORE.
1. Members Detail. ID, Name, DOB, Gender, Type, Email Address, Active, etc.
2. Member's List. An XLS style grid showing a full list of all members. Filters are available.
3. Member's Chart. Shows a visual representation of a member's performance for a set distance and stroke. This chart can be printed.
4. Member's History. Session Date, Distance, Stroke, Time. The member's personal best for each stroke, etc.
5. Member's reports. Detailed reports for current selected member and summary reports for all members.
6. Find a member. Use this dialogue to find a member.
Navigator. A special control use to move through the database records.
Buttons. Find Member, Goto Membership Number or ID, Close Members Dialogue.
