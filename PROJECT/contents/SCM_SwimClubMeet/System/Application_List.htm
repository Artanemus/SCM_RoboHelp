SCM Application List

Special note : During development - build versions may differ. Not all applications shown here may appear in each build.

SwimClubMeet SCM

SwimClubMeet.exe

The core application to administer your members and meets.

Applications bundled with SCM

Help for SCM

Not technically an application. More of a collection of HTML pages. It has full indexing and search features. Written with Adobe RoboHelp.

SCM_BuildMeAClub.exe

Build me a club. It creates the SwimClubMeet database on your MS SQLEXPRESS. It creates all the tables and references, places essential data into those tables, builds SCM functions and sets up logins. There is a unique BMAC constructed for each new version of SCM.

SCM_UpgradeDB.exe

Checks the current SCM MSSQL database version and upgrades the database to match the version in the current SCM installation package. It's capable of performing multi-upgrades, if needed. Upgrades performed include table and field changes to MSSQL SCM, registry changes, preference file inclusion and changes and updates to the support SQLiteV3 databases. Only use this utility when requested by the developer.

FDAdministrator.exe

FireDAC Administrator is a tool to manage SCM internal FireDAC connection definitions. Only use this tool when requested by the developer.

Applications you must install to run SCM,

Microsoft's SQL server Express.

Microsoft SQL Server Express is a version of Microsoft's SQL Server relational database management system that is free to download, distribute and use. It comprises a database specifically targeted for embedded and smaller-scale applications. It's free.

Applications you are advised to install.

Microsoft's Server Management Studio. (MSSMS)

SQL Server Management Studio is an integrated environment for managing any SQL infrastructure, from SQL Server to Azure SQL Database. SSMS provides tools to configure, monitor, and administer instances of SQL Server and databases. Use SSMS to deploy, monitor, and upgrade the data-tier components used by your applications, as well as build queries and scripts. It's free.

Standalone Applications not bundled with SCM

The following applications have been written in a cross platform compiler. It's intended that in future releases there will be Linux, Mac, iPhone, Android and iPad variants of these applications. When you share your database server over wi-fi, these remote applications collectively perform most of the functions that the core application can do.

SCM_Nominate.exe

Beta. This is a stand alone application. This application allows members to quickly nominate for the 'nights' events. It's like a kiosk - swimmers enter their membership number and then nominate for the listed events. Set up a couple of tablets at the front gate for quick event nomination.

SCM_Marshal.exe

This is a stand alone application. This application lets your lane marshall to browse events and heats. Displays entrants, their lane numbers and race time (once they have been entered by Admin or Timekeeper) . The marshall can also flag a heat as 'raced'.

SCM_Timekeeper.exe

This application lets your timekeepers track events, heats for their assigned lane. It shows the entrant name, timetobeat and personal best for the assign lane. The timekeeper may enter the race time for the entrant which is then sent to the server. Or set the roaming feature on. This enabled one Grand Timekeeper Official to collect each lane's time for each heat run. SCM_LeaderBoard, Beta. This is a windows 10 only, stand alone application. It uses a local SQLiteV3 to optimize performance and remove the load off the core MSSQL Server. This application allows you to track member, house and event scores, in real time. It includes a report system. Use this application to track scores for your end-of-season finals. May serve as a billboard. SCM_Help. Help for SwimClubMeet. Forever a work in progress.

Other products (in development)

SCM_Member.exe

Edit member's details, check members details and add new members. Useful in delegating tasks.

SCM_LeaderBoard.exe

House and members scores. Visualize winners and losers, etc. Real-time display of events and scores.

SCM_Chart.exe

Stats, graphs and pie charts for your club's annual general meeting.

SCM_Reports.exe

SCM_Pre-Nominate.exe

Auto post members emails with attached 'personalized' data aware PDF nomination forms. Auto import the member's pre-nomination to the Club's next club night.

SCM_Carnival.exe

Club on club, head-to-heats meets with clubs in your local area.
