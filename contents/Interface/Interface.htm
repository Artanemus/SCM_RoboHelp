The Interface

A quick summary of the interface.

Use the tab-sheets to navigate between Session, Nominate and Heats.

Use the Club Members button to create, modify and delete members and their data.

Use the report buttons to view and print reports.

The menu bar gives you access to other functions, including application settings.

SCM's home form with the session tab selected.

  1. Menu Bar
  2. Title Bar with club name and nickname.
  3. Additional control buttons. Member's access and refresh button.
  4. Tab-sheet buttons. Session, Nominate, Heats.
  5. Session table. Each row represents a 'club night'.
  6. Session buttons. Visibility, lock, new, delete, session report.
  7. Event table. The current events for the selected session.
  8. Event buttons. Expand, move up, move down, new, delete, event report.
  9. Status bar.
