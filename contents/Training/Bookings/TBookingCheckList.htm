Booking Check List

OBS Training booking check list

Overview

Held within the TRAINING Booking form. Located on the third tabsheet named Check List, is a list of ordered items that must be fully checked to certify a booking. Used for quality assurance.

Table location and description

Table name is t_TRAIN_BookingCheckLitsDetail

BookingCheckListDetailID is an auto-generate field.

TrainingDetailID is the ID for the course

ShortName is the description text shown on the Check List tabsheet

IsArchived is used to hide the item

SortID sets the order of display
