Enrollee Check List

OBS Training enrollee checklist,

Table location and description

Located in the table named t_TRAIN_CheckListDetail

CheckListDetailID an auto-number field - created by the system

TrainingDetailID the training course ID

ShortName the description of the check list item - shown for each enrollee, as executed by the Check List and Notes button. 

EnableEmail dictates if an email will be dispatched to the enrollee, as executed by the EMAIL button

IsArchived hides the checklist item 

SortID the order the check list is display

EmailNotes an extended text or memo field that uses metacode to build an automatic email repoonse 

RedirectEmail indicates if the email is to be sent to the enrollee or HRID

RedirectHRID the HRID for email re-direction.
